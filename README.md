### 1. Deploy to Railway
1. Push this repo to GitHub
2. Go to railway.app > New Project > Deploy from GitHub
3. Select your repo
4. Railway will auto-detect Node.js and run `npm start`
5. Add a volume/disk if you want sessions to persist

### 2. Pair the bot
1. Open your deployed Railway URL
2. Enter your WhatsApp number with country code e.g. 263771234567
3. Click Generate Pair Code
4. On WhatsApp: Linked Devices > Link with phone number > Enter code

### 3. Add commands
Drop new `.js` files in `/commands`. Use the template in `/commands/ping.js`.

**Note:** This uses Baileys, an unofficial WhatsApp library. Use at your own risk. Numbers can be banned by WhatsApp for automation.
